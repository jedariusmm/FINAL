# nupidmesktopai
NUPI - AI Desktop and Mobile Application

## iOS App

This repository contains an iOS application built with SwiftUI.

### Features

- Modern SwiftUI-based interface
- Native iOS experience
- AI-powered assistant functionality
- Secure and fast performance

### Project Structure

```
NUPIApp/
├── NUPIApp.xcodeproj/        # Xcode project file
└── NUPIApp/                  # Source code
    ├── NUPIAppApp.swift      # Main app entry point
    ├── ContentView.swift     # Main view
    ├── Assets.xcassets/      # App assets and icons
    └── Preview Content/      # SwiftUI preview assets
```

### Requirements

- iOS 17.0 or later
- Xcode 15.0 or later
- Swift 5.0 or later

### Building the App

1. Open `NUPIApp/NUPIApp.xcodeproj` in Xcode
2. Select your target device or simulator
3. Press Cmd+R to build and run

### App Icon

The app uses the iOS standard app icon format. You can add custom app icons by replacing the files in `NUPIApp/Assets.xcassets/AppIcon.appiconset/`.
