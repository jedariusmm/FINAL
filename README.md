# NUPI AI Assistant

NUPI AI Assistant iOS Application

## Overview

NUPI AI Assistant is an iOS application that provides AI-powered assistance on your iPhone or iPad.

## Project Structure

```
NUPIApp/
├── NUPIApp.xcodeproj/        # Xcode project file
└── NUPIApp/                   # Source code directory
    ├── AppDelegate.swift      # Application delegate
    ├── SceneDelegate.swift    # Scene delegate for iOS 13+
    ├── ViewController.swift   # Main view controller
    ├── Main.storyboard        # Main UI storyboard
    ├── LaunchScreen.storyboard # Launch screen
    ├── Assets.xcassets/       # Asset catalog
    └── Info.plist            # App configuration
```

## Requirements

- iOS 15.0+
- Xcode 14.0+
- Swift 5.0+

## Getting Started

1. Clone the repository
2. Open `NUPIApp/NUPIApp.xcodeproj` in Xcode
3. Select your target device or simulator
4. Press `⌘ + R` to build and run

## Features

- Native iOS interface
- Clean and modern UI design
- Support for iPhone and iPad
- Portrait and landscape orientations
